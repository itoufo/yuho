# MirAI-POST

MirAI-POSTは、伊東雄歩の人格・思想・実績・再現性を
一元化するPR基盤です。ここから指示すれば、
再現性の高いPR/提案/登壇原稿が出せる状態を目指します。

## 構成
- persona.md: 世界観/人格/禁止事項
- proofs.json: 実績DB（数値と根拠）
- decision-patterns.md: 意思決定/再現性のパターン
- output-templates/: 出力テンプレ

## 使い方（例）
- proofs.jsonから用途に合う実績カードを選ぶ
- decision-patterns.mdの型に沿って語る
- output-templatesのテンプレに落とす
