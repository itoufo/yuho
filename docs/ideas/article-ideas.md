## 案1: 社会不適合 × AI（メイン軸）

> ターゲット: 会社員に向いてない人、ADHD、起業志望者
> トーン: 共感ベース、親しみやすい、失敗談を隠さない

### 自己紹介・ストーリー系
- [ ] 訴訟2回、2000万損失、役員の裏切り——それでも僕が「世界征服」を諦めない理由
- [ ] 敬語が使えない、遅刻癖、ADHD——「社会不適合」の僕が経営者になれた理由
- [ ] IQ130超えの孤独——頭が良すぎて友達ができなかった話
- [ ] 「UFO! UFO!」といじめられた僕が、名前を誇りに変えるまで
- [ ] 会社員2年で限界だった僕が、いきなり株式会社を作った理由

### ADHD × 仕事術
- [ ] ADHDでも経営できる——僕が10年会社を潰さなかった方法
- [ ] 遅刻癖、忘れ物癖、敬語が使えない——それでも信頼される方法
- [ ] ADHDの「過集中」を武器にする方法
- [ ] 「普通」ができない人のための仕事の選び方
- [ ] ADHDが起業に向いている3つの理由

### AI × キャリア
- [ ] 「AIに仕事を奪われる」のではなく「AIを味方につける」人生を選んだ話
- [ ] AIを「こんにちは」から始めた僕が、AI教育の専門家になるまで
- [ ] 社会不適合者こそAIを学ぶべき理由
- [ ] 3年かかるプログラミング学習を2日に短縮した方法
- [ ] AIフレンズ継続率90%の秘密——なぜ人は学びを続けられないのか

### 起業・経営リアル
- [ ] フリーランスを経由せず、いきなり株式会社を作った理由
- [ ] MENSAメンバーだけでチームを作ったら大失敗した話
- [ ] 10代専用SNSを作って、マネタイズに失敗した話
- [ ] 「AIなんだから100点でしょ？」——顧客との認識ギャップ地獄
- [ ] 本人訴訟で弁護士事務所に勝った話
- [ ] 役員に顧客を奪われて訴訟した話
- [ ] 上位企業がトンズラして2000万円消えた話
- [ ] オフショア開発でトラブル頻発、ブリッジSEが失踪した話
- [ ] 150人月の大規模プロジェクトを生き延びた方法

### 学習・成長
- [ ] 失敗しまくった大学時代が、今の教育メソッドの原点になった話
- [ ] SEOで「恋愛術」「大学受験 参考書」でGoogle1位を取った方法
- [ ] JDLA認定講座の講師になるまでの道のり
- [ ] 心理学・教育学に1000万円投資して学んだこと

## 案2: 哲学者モード（上級者向け）

> ターゲット: 思考が深い人、本質を求める人
> トーン: 淡々、こびない、読む人を選ぶ

### 常識への疑問
- [ ] 「努力すれば報われる」は本当か？
- [ ] なぜ学校教育は「点」しか教えないのか
- [ ] 「頭がいい」とは何か——IQ130超えの僕が考える知性の正体
- [ ] 資格を取る意味は本当にあるのか
- [ ] 「AIに聞けばいい時代」に、人間が学ぶ意味とは
- [ ] 「常識」は誰が決めているのか
- [ ] なぜ「普通」を目指すと苦しくなるのか

### 世界の見方
- [ ] 世界は「点」で見ると狭い、「体積」で見ると広がる
- [ ] 情報は「心のフィルター」を通って脳に届く
- [ ] 人は見たいものしか見えない——認知バイアスの話
- [ ] 「失敗」と「経験」の違いは、捉え方だけ
- [ ] 期待値が感情を決める——「当たり前」の心理学
- [ ] 坂本龍馬が教えてくれた「ヒーロー」の定義

### 成長力学
- [ ] 成長力学入門——知識を「点」から「体積」に変える方法
- [ ] なぜ知識だけでは成長しないのか
- [ ] 記憶は「効率」「尊厳」「目標達成」で最適化される
- [ ] アイデアが生まれる本当の条件——既存知識×リラックス
- [ ] 「体積」の大きい人はなぜ発想力が高いのか
- [ ] リラックスの前に「知識の体積化」が必要な理由
- [ ] ニュートンがリンゴを見て万有引力を発見できた本当の理由
- [ ] 学習の「フィードバックループ」を回す方法

### 心のエネルギー論
- [ ] あらゆる知的行動は、心が原動力となって稼働する
- [ ] 心のエネルギーは「外」と「内」から来る——内側の方が質がいい
- [ ] 適切な目標設定が内部エネルギーを生む理由
- [ ] モチベーションが続かない本当の理由
- [ ] 「やる気」に頼らない行動設計

### NLP（神経言語プログラミング）
- [ ] 言葉が思考を作り、思考が行動を作る
- [ ] 無意識にアプローチする技術
- [ ] 「できない」という言葉が本当にできなくさせる理由
- [ ] 行動変容のメカニズム——なぜ人は変われないのか
- [ ] 自分を変えるための「リフレーミング」入門

### ヒーロー論・人生哲学
- [ ] 「世界征服」の本当の意味
- [ ] ヒーローとは「世界に影響を与える者」である
- [ ] 「雄歩」——英雄然として歩む、という名前の意味
- [ ] 現代のヒーローは経営者か、それとも別の何かか
- [ ] 1秒の価値——自分の時間を誰のために使うか
- [ ] 「失敗は経験の元」を座右の銘にした理由
